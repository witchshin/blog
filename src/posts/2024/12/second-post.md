# 두 번째 포스트 - Vite 사용하기

Vite는 현대적인 프론트엔드 빌드 도구입니다.

## Vite의 장점

- **빠른 개발 서버**: HMR(Hot Module Replacement) 지원
- **최적화된 빌드**: Rollup 기반의 효율적인 번들링
- **TypeScript 지원**: 별도 설정 없이 바로 사용 가능

## 설치 방법

```bash
npm create vite@latest my-app -- --template react-ts
```

## 사용 예시

Vite는 매우 간단하게 사용할 수 있습니다:

1. 프로젝트 생성
2. 의존성 설치
3. 개발 서버 실행

> "Vite는 개발 경험을 크게 향상시킵니다." - 개발자들

## 결론

Vite를 사용하면 더 빠르고 즐거운 개발을 할 수 있습니다!
