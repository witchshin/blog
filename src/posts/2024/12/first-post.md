# 첫 번째 포스트

안녕하세요! 이것은 첫 번째 블로그 포스트입니다.

## 소개

이 블로그는 **React**와 **TypeScript**를 사용하여 만들어졌습니다. 마크다운으로 글을 작성하고, 자동으로 HTML로 변환됩니다.

## 주요 기능

1. 마크다운 지원
2. 태그 및 카테고리 분류
3. 월별 포스트 관리
4. 반응형 디자인

## 코드 예시

function greet(name: string) {
  return `Hello, ${name}!`;
}
